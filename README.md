Open with VS2015/2017 and compile

## Usage

Update Settings.txt as needed:

* Report Year (Default: current year) = 2017

* Export Spreadsheet Name (Default: EPA.xlsx) = EPA.xlsx


Double click EPA_Automation.exe to run
